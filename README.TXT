========================
BUILD OUTPUT DESCRIPTION
========================

To run the project from the command line, go to the dist folder and
type the following:

java -jar "TheTower.jar" 
--------------------------------------------------------------------------------------------------------
If the RUNGAME.bat is not avaliable:

- Please create a new text document.
- Rename it to RUN.bat and click yes to accept that the file gets a name change and extension change.
- Right click and click the edit file. 
- Then add the above code. 
- Once done save and close the document.
